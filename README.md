# Simple chess dragging

A tiny toy project.
Simply move a chess piece freely on a 2x2 board.

[simple_chess_dragging.webm](https://github.com/user-attachments/assets/27653a69-b3fa-4c92-83b4-dd1a4eb077e7)


Built with Rust and Gtk4-rs.
